# Manga Library

## Описание
Веб-приложение для управления личной библиотекой манги.  
Пользователи могут добавлять мангу, просматривать коллекции, искать по названию, автору или жанру.  
Простая и удобная система для энтузиастов манги.

---

## Функционал

### Для **пользователей**
- Регистрация и вход
- Добавление манги в библиотеку
- Просмотр личной коллекции
- Поиск по названию, автору, жанру
- Детальная страница манги (тома, главы, обложка)

### Для **администраторов**
- Управление базой манги
- Модерация пользовательских загрузок
- Статистика библиотеки

---

## Типы данных
- **Manga** — название, автор, жанр, обложка
- **Volume** — номер тома, дата выхода
- **Chapter** — номер главы, страницы
