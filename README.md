Manga Library
Описание
Веб-приложение для управления библиотекой манги.
Пользователи могут добавлять мангу, просматривать коллекции, искать по категориям и авторам.
Это простая система для энтузиастов манги, с базовым функционалом каталога.

Функционал
Для пользователей

Регистрация и вход в систему
Добавление манги в библиотеку
Просмотр личной коллекции
Поиск по названию, автору или жанру
Просмотр деталей манги (тома, главы, обложка)

Для администраторов

Управление базой манги (добавление/удаление)
Модерация пользовательских загрузок
Статистика по библиотеке
